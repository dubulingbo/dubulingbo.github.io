快捷键 - 搜集比较常用的快捷键

发表于 2022-09-03 本文字数 2.5k 字 阅读时长 2 分钟
  • Habitplus の ブログ
首页 效率办公

# 一、Windows 10 快捷键

快捷键说明
Windows打开开始菜单栏
Windows + U打开设置
Windows + D回到桌面
Windows + Tab桌面切换
Windows + R打开命令窗口（CMD）
Windows + E打开文件资源管理器
Windows + S全局搜索
Windows + A打开右侧多任务侧边栏
Windows + L锁屏
Windows + V打开剪贴板历史记录
Windows + "+/-"放大 / 缩小屏幕
Windows + Ctrl + "←/→"快速切换虚拟桌面
Alt + Tab窗口切换

# 二、Chrome 快捷键 --- Windows

快捷键操作描述
Ctrl + n打开新窗口
Ctrl + Shift + n以隐身模式打开新窗口
Ctrl + t打开新的标签页，并切换到该标签页
Ctrl + Tab切换到下一个标签页（向右平移一个）
Ctrl + Shift + Tab切换到上一个标签页（向左平移一个）
Ctrl + 1 到 Ctrl + 8切换到指定标签页
Ctrl + 9切换到最后一个标签页（最右边一个）
Ctrl + w关闭当前标签页
Ctrl + f在当前页面进行内容搜索
Ctrl + Shift + w / Alt + F4关闭所有标签页和浏览器
Ctrl + h在新标签页中打开 “历史记录”
Ctrl + j在新标签页中打开 “下载内容”
Ctrl + Shift + Del在新标签页中打开 “清除浏览数据” 弹窗
Ctrl + Shift + b显示或隐藏书签栏
Ctrl + Shift + o在新标签页中打开 “书签管理页面”
Alt + Home在当前标签页中打开 “主页”
Alt + d | F6在当前标签页，跳转到地址栏
Ctrl + R重新加载当前网页，相当于按下 “刷新 “ 按钮
Ctrl + Shift + R重新加载当前网页（忽略缓存的内容）
Ctrl + U在新标签页中查看此页面源码
Ctrl + D将当前网页保存为书签
F11开启或关闭全屏模式
Ctrl + +放大网页上的所有内容
Ctrl + 0将网页上的所有内容恢复到默认大小
Ctrl + -缩小网页上的所有内容

# 三、IDEA 快捷键 --- Windows

快捷键说明
Ctrl + D复制光标所在行或已选择的部分
Ctrl + Y删除一行
Ctrl + Z撤销
Shift + F6重命名（文件名、类名、变量名）
Ctrl + Ctrl全局搜索类
Ctrl + F当前文件中搜索
Ctrl + R当前文件中替换
Ctrl + Shift + U全部转换成大写字母
Ctrl + Shift + L全部转换成小写字母

# 四、VsCode 快捷键

快捷键描述
Ctrl + X剪切一行或选择的
Ctrl + C复制一行或选择的
Alt + ↑/↓把光标所在行上移 / 下移
Shift + Alt + ↑/↓把光标所在行往上 / 下复制一行
Ctrl + B显示或隐藏侧边栏

方向键移动图形
Ctrl + G | Ctrl + Shift + G组合

Ctrl + C复制
Ctrl + V粘贴

Ctrl + A全选图形

Ctrl + F查找文本
Ctrl + T <br />Ctrl/Shift + Q清除文本
Ctrl + Shift + A全部大写
Ctrl + Shift + K小型大写字母
Ctrl + N新建文件
Ctrl + P <br />Ctrl + F2 <br />Ctrl + Shift + F12打印文件
Ctrl + F4 <br />Alt + F4 <br />Ctrl + Shift + F4 <br />Ctrl + W关闭文件
Ctrl + 鼠标滚轮上滚：放大视图 <br /> 下滚：缩小视图
Shift + 鼠标滚轮上滚：左移视图 <br /> 下滚：右移视图
鼠标滚轮上下移动视图

Ctrl + Alt + 鼠标右键单击 <br />Alt + Shift + F6缩小画面

Ctrl + Alt + 鼠标左键框选放大选中的画面部分
