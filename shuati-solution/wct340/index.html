# T2. 等值距离和

链接：https://leetcode.cn/problems/sum-of-distances/

# 解题思路：HashMap + 累加前缀和

  • 将数组中的相同值的下标分成一组
  • 计算每一组内的各个下标的距离之和，这个可以使用递推进行计算，减少时间复杂度：
    • dp[i] = dp[i-1] + (i-1) * (arr[i] - arr[i-1]) ， dp[i] 表示 0, 1, ... i-1 到 i 的距离之和
